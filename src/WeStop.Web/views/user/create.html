O que é STOP?

Adedonha!, Adedanha! ou Stop! é um jogo que consiste em desenhar-se uma tabela em tópicos em um papel para cada jogador. Cada coluna da tabela recebe o nome de uma categoria de palavras como animais, automóveis, nomes pessoais, cores etc., e cada linha representa uma rodada do jogo.

A cada rodada uma letra é sorteada. Os jogadores então devem preencher todas as colunas com palavras que iniciam com essa letra. Quando um jogador preencher todas as colunas, ele pode chamar STOP, fazendo com que todos os outros parem imediatamente de preencher suas colunas.

É calculada então a pontuação de cada jogador, sendo que respostas repetidas valem 5 pontos, respostas não repetidas valem 10 pontos e respostas erradas ou em branco valem 0 pontos.

Para definir um vencedor, deve-se somar todas as pontuações dos jogadores em todas as rodadas, quem obtiver o maior valor, é o vencedor!
